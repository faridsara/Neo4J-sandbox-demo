# Neo4J-sandbox-demo

Neo4j sandbox demo with scala. 

# Dependencies

- [java library provided by neo4j](https://github.com/neo4j/neo4j-java-driver)
- [cats effect dependencies](https://github.com/typelevel/cats-effect)
